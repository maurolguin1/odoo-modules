Stock picking nomenclature

Generate name of lot or serial number during picking operation

Automatic Lot Name

During the picking operation of purchased products, you can easily setup lot with inline editing. This module improve this feature with enabling simple nomenclature for pre-populate lot name's field.

You can use vendor reference and/or product code as a prefix. If reference or code are not defined, a slugify version of the name is used.

Vendor slugify name and product code

Suffix append after picking validation

In module settings, you can also setup a sequence to use for append a suffix in your lot name. This append after you validate the picking operation ; then you can use a non_gap sequence if you need.

Automatic Serial Number

During the picking operation of purchased products with serial, a serial number is automatically generated according to your configuration settings.

This serial number can contain vendor reference, product reference and sequence number depending of your settings.

Serial number generation

Settings

Module settings are available in Inventory > Settings ; only if Lots and Serial Numbers tracking is activate.

Information

This module requires python-slugify Python package. You can install it using pip: pip install python-slugify
