Partner Reference Sequence

Setup sequence for partner internal reference

When create a partner, if no internal reference is defined by the user, an internal reference is picked from a sequence depending of the partner type : vendor or client.

If the partner is a vendor, we use the next sequence of vendor as internal reference ; even if it also a client. If the partner is a client, we use the next sequence of client as internal reference.

Note: Only company partner are concerned by this module, other res.partner are exclude from the scope of this module.
