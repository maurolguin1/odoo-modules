Product Name Reference

Override product name_get method

The default name_get method of product.template and product.product return the reference and the name of the product: [reference] Product name.

With this module, the name_get method only return the product reference.

Product reference + name | Product reference only

Notes

Kanban view of products show explicitly the name of the product with the reference ; this module doesn't handle modification of Odoo View, then after install it the Kanban view of products still the same.
