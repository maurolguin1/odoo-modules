===========================
Sale Order Customer Comment
===========================

Add a new field in Sale Order for the customer comment ; this module is very basic and
it is intended to be used with XML RPC API or other module that extend it.

For details, customization or feedback please contact us: contact@critech-services.com

No external library required.
