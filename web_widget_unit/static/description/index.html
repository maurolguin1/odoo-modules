Web Widget Unit

New widget option for display field with unit like price per sqm

This module add new widget options for display field with unit like volume field, weight field, price per unit, etc.

It work with field unit like product_uom or width static unit provided by a simple string value.

Usage

Use with static string unit

<field name="weight"
       options="{
            'currency_field': 'currency_id',
            'unit': {
                'text': 'kg',
                'separator': ' / '
            },
       }"
/>

Use with field unit

<field name="uom_so_list_price"
       widget="monetary"
       options="{
            'currency_field': 'currency_id',
            'unit': {
                'field': 'uom_so_id',
                'model': 'product.uom',
                'model_label': 'display_name',
                'separator': ' / '
            },
       }"
/>
